# Pong Clone

A classic Pong game recreated in Unity 6 using C#.

## Features
- Two-player paddle controls
- Physics-based ball movement
- Increasing ball speed after paddle hits
- Score tracking
- First player to 10 wins
- Restart game with R
- Clean gameplay architecture

## Controls

Player 1
W - Up
S - Down

Player 2
Up Arrow - Up
Down Arrow - Down

R - Restart after game over

## Built With

- Unity 6
- C#
- TextMesh Pro
- Rigidbody2D Physics
